# Лекция по ИИ и промпт-инжинирингу

В этой папке собран готовый комплект материалов:

- `index.html` — статическая веб-презентация, подходящая для GitHub Pages;
- `materials/lecture.md` — текстовая версия доклада;
- `materials/run-of-show.md` — тайминг и сценарий выступления на 60-90 минут;
- `materials/prompt-examples.md` — практические примеры промптов для разбора;
- `materials/sources.md` — опорные источники и ссылки;
- `assets/` — стили, скрипт и SVG-визуалы.

## Как открыть локально

Достаточно открыть `index.html` в браузере.

По умолчанию презентация открывается в широком режиме, удобном для большого экрана.

Переключение режимов:

- кнопка `Сайт` / `Экран` в верхней панели;
- клавиша `F` — быстрый переход между режимами.

Если нужен локальный сервер:

```powershell
python -m http.server 8000
```

После этого открыть:

`http://localhost:8000`

## Как выложить на GitHub Pages

1. Загрузить содержимое этой папки в репозиторий.
2. В настройках репозитория включить GitHub Pages.
3. В качестве источника выбрать ветку и корень репозитория либо папку `docs`, если позже захотите перенести сайт туда.
4. Убедиться, что `index.html` лежит в выбранном корне публикации.
