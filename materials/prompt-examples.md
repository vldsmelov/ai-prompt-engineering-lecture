# Практические примеры промптов для лекции

## 1. Объясняющий текст

Плохо:

`Напиши текст про ИИ.`

Лучше:

`Подготовь понятный текст для офисных сотрудников о различиях между ИИ-чатом, поисковым ИИ и ИИ-агентом. Объем 700-900 слов. Тон спокойный, без жаргона. Добавь 3 бытовых примера и чек-лист выбора инструмента.`

## 2. Суммаризация документа

Плохо:

`Сократи этот документ.`

Лучше:

`Суммируй документ для руководителя, у которого есть 2 минуты на чтение. Выдели 5 ключевых идей, 3 риска и 3 решения. Сохрани деловой тон. Не добавляй фактов вне документа.`

## 3. Анализ встречи

Плохо:

`Разбери стенограмму.`

Лучше:

`Проанализируй стенограмму встречи. Верни: 1) принятые решения, 2) открытые вопросы, 3) ответственных, 4) дедлайны, 5) риски недопонимания. Если в стенограмме чего-то нет, явно напиши, что это не указано.`

## 4. Исследовательский запрос

Плохо:

`Сравни сервисы для аналитики.`

Лучше:

`Собери сравнение сервисов аналитики продаж для среднего B2B-бизнеса. Используй только актуальные источники. Верни таблицу со столбцами: продукт, позиционирование, ключевые функции, наличие API, ориентир по цене, сильные стороны, ограничения. В конце дай короткую рекомендацию для компании из 50-200 человек.`

## 5. Промпт для tool-use системы

Лучше формулировать так:

`Проанализируй приложенный CSV. Сначала проверь структуру файла и опиши столбцы. Затем найди пропуски, выбросы и дубликаты. После этого посчитай основные показатели по каждому сегменту. Верни итог в виде краткой аналитической записки и таблицы. Если для вычислений нужен код, используй его, а не считай вручную.`

## 6. Промпт для генерации письма

`Подготовь письмо клиенту по итогам пилота. Аудитория: директор по цифровой трансформации. Тон: деловой, уверенный, без агрессии. Цель: зафиксировать результаты пилота и предложить следующий этап. Структура: 1) что сделали, 2) какой эффект увидели, 3) что предлагаем дальше, 4) призыв к короткому созвону. Объем: до 180 слов.`

## 7. Промпт для уточнения

Когда задача еще сырая, полезно не требовать сразу финальный ответ, а попросить модель помочь собрать постановку:

`Я пока не до конца сформулировал задачу. Помоги превратить мою идею в хороший запрос к ИИ. Сначала задай мне 5 самых полезных уточняющих вопросов, затем на основе моих ответов собери финальный промпт.`
